Quickstart

Get your AI reading emails in under 5 minutes

1 Create an account

Sign up at app.moperator.work to get your API key and email address.

2 Query emails via REST API 

# List all emails
curl https://api.moperator.work/api/v1/emails \
  -H "Authorization: Bearer mop_yourkey"

# Filter by label
curl "https://api.moperator.work/api/v1/emails?labels=finance" \
  -H "Authorization: Bearer mop_yourkey"

# Get unread emails only
curl "https://api.moperator.work/api/v1/emails?status=unread" \
  -H "Authorization: Bearer mop_yourkey"

3 Or connect Claude Desktop via MCP

Add to your claude_desktop_config.json:

{
  "mcpServers": {
    "moperator": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://api.moperator.work/mcp",
               "--header", "Authorization: Bearer mop_yourkey"]
    }
  }
}

Then ask Claude: "Check my email" or "Do I have any finance emails?"
